# Sudoku Game

## Overview
I have played Sudoku almost daily for the past decade, and this project is a way to turn a personal hobby into a complete, full-stack application. It includes:
  - A Java backend for puzzle generation, validation, and solving.
  - A frontend (HTML/CSS/JS...for now, possible upgraded later) for an interactive playing experience
  - Support for multiple difficult levels.

---

## Features
  - Generate valid Sudoku puzzles
  - Play puzzles in the browser
  - Check correctness and validate moves
  - Difficulty levels: Easy, Medium, Hard, Expert
  - Automatic Solver (planned)
  - Hint system (planned)
  - Timer and statistics (planned)

---

## Tech Stack
### Backend
  - Java (17+ recommended)
